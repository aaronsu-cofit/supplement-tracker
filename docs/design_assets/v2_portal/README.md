本資料夾包含了我們與產品團隊初步凝聚共識的「遊戲化 (Gamification) + 毛玻璃 (Glassmorphism)」參考設計圖。

## 📍 目錄結構與檔案說明

本資料夾內包含以下 AI 生成的參考圖檔，請設計師以此作為視覺風格與排版的基底：

1. **`1_web_portal_reference.png`** 
   - **用途：** Web 端入口大廳 (Dashboard) 的排版參考。
   - **說明：** 採用純白極簡底色，2x3 六宮格卡片設計。請特別留意圖示「骨骼」需保持乾淨無肌肉，且「荷爾蒙」採用水滴圓潤意象。
   
2. **`2_line_rich_menu_reference.png`**
   - **用途：** LINE 官方帳號底部的「圖文選單 (Rich Menu)」視覺參考。
   - **說明：** 特點為發光描邊 (Neon Glow)、高度點擊暗示。

3. **`3_style_inspiration_glassmorphism.png`**
   - **用途：** 材質與色彩靈感庫。
   - **說明：** 團隊特別喜歡這張圖中的「半透明毛玻璃材質」與「高飽和度但柔和的漸層底色」，請在後續的元件設計中參考此種光影處理方式。

## 🎨 設計師協作指南 (Action Items)

為了讓工程端能順利將您的設計無縫實裝，麻煩您協助產出以下兩種規格的圖檔/切圖：

### 1. LINE 圖文選單 (Rich Menu) 正式圖檔
- **尺寸要求：** 必須剛好是 **`2500 x 1686` 像素** (寬x高)。
- **切分方式：** 請在畫布上精準切割為「2 列 x 3 欄」的六宮格，每一格的大小應為 `833 x 843` 像素（中間的欄位寬度可微調為 `834` 以補齊 `2500`）。
- **格式：** `JPEG` 或 `PNG`（檔案大小須 `< 1MB`）。
- **交付：** 提供一張完整合併的單一圖片即可，工程端會透過 API 自動切分觸發熱區。

### 2. Web App 獨立圖示 (Icons)
- **用途：** 網頁大廳卡片上的 3D / Q版圖示。
- **清單：** 
  1. 保健品 (Supplements)
  2. 傷口照護 (Wounds)
  3. 骨骼關節 (Bones)
  4. 性健康 (Intimacy)
  5. 荷爾蒙健康 (Hormones)
  6. 小習慣打卡 (Habits)
- **尺寸要求：** 每張獨立匯出為 **`512 x 512` 像素** 的透明背景。
- **格式：** 背景全透明的 `PNG`，或 `SVG`、`.lottie` 動畫檔尤佳。
- **交付：** 請上傳至 Figma，並直接 Tag 工程團隊 (`@johnson`) 取用。
